🔥 Firebase Setup
Connect your live database
All data — assets, employees and history — is stored in Firebase and syncs live across every device and team member. Enter your Firebase config below once and it's saved permanently in this file.
⚠️ Important — Firebase Rules: If you can't add assets or employees, your Firebase rules may have expired. Go to Firebase Console → Realtime Database → Rules and set them to: 
{
  "rules": {
    ".read": true,
    ".write": true
  }
}
Then click Publish.
