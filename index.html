🔥 Firebase Setup
Connect your database
Firebase is Google's free real-time database. All your team members will see live updates the moment anyone makes a change. Follow these steps — it takes about 5 minutes.
1
Create a Firebase project
Go to console.firebase.google.com → Add project → give it a name (e.g. "AssetTrack") → Continue. Disable Google Analytics if you don't need it → Create project.
2
Create a Realtime Database
In the left sidebar → Build → Realtime Database → Create Database → choose your region → Start in test mode (allows read/write for 30 days — you can secure it later) → Enable.
3
Get your config keys
Project Settings (gear icon) → General → Your apps → click Web icon (</>) → Register app → copy the firebaseConfig values below.
4
Paste your config and connect
Fill in the fields below, click Connect, and your database is live.
